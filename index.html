Selecting a date was never easier

Vuelendar is fully-interactive and allows you to manage events freely. You can specify various series of selections and style them with SCSS easily.

See Documentation

Usage

  1. Install via NPM

    Install package as a dependency in your project. 

    npm i vuelendar
    or 
    yarn add vuelendar

  2. Load in your code

    JS

    Import and register Vuelendar component, 

    import VDaySelector from 'vuelendar/components/vl-day-selector';
export default {
  components: {
    VDaySelector
  },
  data () {
    return { date: null }
  }
}

    Use Vuelendar in a template. 

    <v-day-selector v-model="date" />

    CSS

    Import styles in vue file 

    <style src="vuelendar/scss/vuelendar.scss" lang="scss"></style>
    or in your SCSS 
    @import '~vuelendar/scss/vuelendar.scss';

  3. Extend as you wish

    You can customize Vuelendar by overriding SCSS variables to fit your app. 

    // Import Vuelendar stylesheet
@import '~vuelendar/scss/vuelendar.scss';

// Set Vuelendar primary color to primary color of your app
$vl-primary: $color-primary;
