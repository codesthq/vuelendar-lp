Introduction

Simple and clean calendar written in Vue.js.

Features

Select single date

Select range of dates

Requirements

  • Vue.js

Demo

Go to demo page .

Installation

Install package 

npm install vuelendar@1.0.0

Include calendar styles 

<style src="vuelendar/scss/vuelendar.scss" lang="scss"></style>

or in your SCSS 

@import '~vuelendar/scss/vuelendar.scss';

Types

DateString

Represents date as a string.
Is used internally in vuelendar, and is represented by date YYYY-MM-DD format.

YYYY-MM-DD format is has it's practical benefits since it can be easily compared.

FunctionObject

Object that should consist of key-function pairs.

Example: 
{
  'example-class': date => date > '2010-02-20',
  'event-class': date => date === '2020-01-01',
}

Day selector

Usage

Register component

import VDaySelector from 'vuelendar/components/vl-day-selector';
export default {
  components: {
    VDaySelector
  },
  data () {
    return { date: null }
  }
}

Use in template

<v-day-selector v-model="date" />

Props

selected-date
DateString
Currently Selected date.
custom-classes
FunctionObject

Defines custom classes. Those will be added to a day input (.vl-calendar-month__day).

An object should consist of key-function pairs, where key defines a class to be added and function should return Boolean weather class should be added or not. Function get date in DateString as an argument.

default-date
DateString

Based on this date active month and year are calculated. 

<v-day-selector
  v-model="date"
  :default-date="'2020-01-01'"
/>

In this example active month will be January and active year 2020.

is-disabled
Function DateString Boolean

A function that should decide if a given date should be disabled. 

<v-day-selector
  v-model="date"
  :is-disabled="date => date > '2020-01-01'"
/>

In this example dates later than 1st of January 2020 will be disabled.

single-month
Boolean
Render only one month pane.
first-day-of-week
String

First day of the week based on day short name.

Default: mon

Range selector

Usage

Register component

import VRangeSelector from 'vuelendar/components/vl-range-selector';
export default {
  components: {
    VRangeSelector
  },
  data () {
    return { range: {} }
  }
}

Use in template

<v-range-selector
  :start-date.sync="range.start"
  :end-date.sync="range.end"
/>

Props

start-date
DateString
Start of the selected range.
end-date
DateString
End of the selected range.
custom-classes
String

Defines custom classes. Those will be added to a day input (.vl-calendar-month__day).

The object should consist of key-function pairs, where key defines a class to be added and function should return Boolean weather class should be added or not. Function get date in DateString as an argument.

default-date
DateString

Based on this date active month and year are calculated. 

<v-day-selector
  :start-date.sync="range.start"
  :end-date.sync="range.end"
  :default-date="'2020-01-01'"
/>

In this example active month will be January and active year 2020.

block-start-date
Boolean
Block change of start date - control only end date.
disabled
Boolean
Disable input.
is-disabled
Function DateString Boolean

A function that should decide if a given date should be disabled. 

<v-range-selector
  :start-date.sync="range.start"
  :end-date.sync="range.end"
  :is-disabled="date => date > '2020-01-01'"
/>

In this example dates later than the 1st of January 2020 will be disabled.

single-month
Boolean
Render only one month pane.
enable-single-date
Boolean
Enables the possibility to select the same start and end date
first-day-of-week
String

First day of the week based on day short name.

Default: mon
